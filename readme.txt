WP Yamli allows blog editors and commenters to write Arabic spelling text in English or French and Yamli will convert them.

== Description ==

WP Yamli is a simple and customizable plugin that enables blog editors and commenters to use Yamli service in the post editor and comments fields. with Yamli you can "simply spell out each Arabic word the way you would pronounce it, and Yamli will convert it to the corresponding Arabic word using its patent pending technology. Yamli is the web's best solution for Arabic transliteration." Whether you, or your commenters don't have Arabic keyboard, you can simply use this plugin to simplify your life.

== Installation ==

1. Put yamli folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface, plugins page and activate the plugin
3. You should see a new option "Yamli" in the settings menu.
